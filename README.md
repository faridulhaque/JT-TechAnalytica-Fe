# 🗂️ Task Management App (Frontend)


## 🚀 Overview

A simple task management app where users are assigned tasks by an admin, can update their status, and admins can assign tasks, make updates, and track activity logs.



## 🧰 Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Daisy UI

### Backend
- NestJS
- TypeORM
